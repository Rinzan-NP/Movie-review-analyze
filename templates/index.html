Model Performance Analysis

Model Accuracy Comparison

This chart compares the accuracy of different machine learning models on the test dataset. Higher accuracy means the model correctly predicts sentiment more often. The comparison helps us understand which algorithm performs best for sentiment analysis tasks.

AUC Scores Comparison

The Area Under Curve (AUC) score measures how well each model can distinguish between positive and negative sentiments. A higher AUC score indicates better model performance. AUC values closer to 1.0 represent excellent classification ability.

Precision, Recall & F1-Score Comparison

This comprehensive comparison shows three key performance metrics side by side. Precision measures how many predicted positives were actually positive. Recall measures how many actual positives were correctly identified. F1-Score is the harmonic mean of precision and recall, providing a balanced view of model performance.

Performance Radar Chart

This radar chart provides a comprehensive view of all model performance metrics in one visualization. Each model is represented by a colored shape, and the area covered indicates overall performance. Models with larger areas across all metrics generally perform better across different evaluation criteria.

All ROC Curves Comparison

This chart shows all ROC curves on a single plot for direct comparison. Curves closer to the top-left corner indicate better performance. The diagonal dashed line represents random classification performance. This visualization helps identify which models perform best across different threshold settings.

Logistic Regression ROC Curve

The ROC curve shows the trade-off between sensitivity and specificity for Logistic Regression. The area under the curve (AUC) indicates how well the model distinguishes between sentiments.

Naive Bayes ROC Curve

This ROC curve evaluates the Naive Bayes classifier's performance. A curve closer to the top-left corner indicates better performance.

Bagging Ensemble ROC Curve

The Bagging ensemble method combines multiple models to improve performance. This ROC curve shows how the ensemble approach affects model performance.

Voting Classifier ROC Curve

The Voting Classifier combines predictions from multiple models using soft voting. This demonstrates how ensemble methods can achieve superior performance.
